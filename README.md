# Dropbox Homepage Clone

## Project Overview
This project is a clone of the Dropbox homepage developed using HTML, CSS, and JavaScript. The website recreates the design and user interface of Dropbox while implementing animations, responsive layouts, multimedia content, and interactive features.

## Features
- Fixed Navigation Bar
- Hero Section
- Animated Logo Slider
- Find Section with Scroll Animations
- Organise Section
- Share Section
- Security Section
- Testimonial Slider
- Industry Cards
- Discover Resources Section
- Footer Navigation
- Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
- HTML for webpage structure
- CSS for styling and animations
- JavaScript for interactivity and dynamic effects

## Learning Outcomes
Through this project, I learned:
- Web page structuring using HTML
- Styling using CSS
- Responsive layouts with Flexbox and Grid
- JavaScript DOM Manipulation
- Event Handling
- Scroll Animations
- Interactive User Interface Design

## How to Run
1. Download the project files.
2. Open `dropbox.htm` in a web browser.
3. Explore the website features and animations.
